# BilimX Course Builder

Современный конструктор онлайн-курсов с интерфейсом в стиле SaaS.

## Как запустить

1. Откройте файл `index.html` в браузере.
2. Для стабильной загрузки CDN-библиотек лучше запускать через локальный сервер.

Пример через Python:

```bash
python -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Что реализовано

- Sidebar с модулями и уроками (раскрытие/сворачивание).
- Выбор урока и подсветка активного элемента.
- Header с названием `BilimX`, шагами урока и общим прогрессом.
- Основная область с редактором блоков: заголовок, текст, блок.
- Создание и удаление модулей/уроков.
- Отметка урока как пройденного.
- Drag & Drop для изменения порядка уроков.
- Автосохранение структуры курса и контента в `localStorage`.
- Адаптивность для desktop и mobile.
